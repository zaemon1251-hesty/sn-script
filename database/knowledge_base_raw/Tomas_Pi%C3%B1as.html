Tomas Piñas Bermudez (born 8 July 1982[1] in Granada) is a class 3 wheel chair using para table tennis player from Spain. In 2012, he lived in Ogijares, Granada. He played table tennis at the 2004, 2008 and 2012 Summer Paralympics. In 2008, he finished third in the class 3 singles table tennis game.[2]
